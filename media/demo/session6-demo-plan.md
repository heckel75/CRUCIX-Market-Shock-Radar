# Session 6 demo plan

Target length: 20-30 seconds

1. Start on http://localhost:3117 and show Crucix is running.
2. Open http://localhost:3117/api/data briefly to show the live source JSON.
3. Switch to Command Prompt and run npm run shock.
4. Show the terminal output: candidates extracted, signals matched, score, regime, JSON written.
5. Switch to http://localhost:3117/market-shock.html.
6. End on the dashboard headline, score, Shock mode regime, and ranked signal board.

Suggested voiceover:
World events move markets before charts explain them. This prototype reads Crucix OSINT data, maps live signals into possible market-risk channels, generates a Market Shock Score, and renders a dashboard for a pre-market risk scan. Experimental only, not investment advice.
